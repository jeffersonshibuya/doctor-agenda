## Doctor Agenda

A Next.js app scaffolded with the App Router. Use it as the starting point for a doctor scheduling/agenda experience.

## Tech Stack

- Next.js (App Router)
- React
- TypeScript
- Tailwind CSS
- ESLint

## Getting Started

Install dependencies and run the development server:

```bash
npm install
npm run dev
# or
yarn
yarn dev
# or
pnpm install
pnpm dev
# or
bun install
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the app.

## Scripts

```bash
npm run dev
npm run build
npm run start
npm run lint
```

## Project Structure

- `app/page.tsx` - main page entry
- `app/layout.tsx` - root layout
- `app/globals.css` - global styles

## Notes

- The page hot-reloads as you edit files in `app/`.
- Fonts are handled via `next/font`.
